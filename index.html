This is the GitHub Activity

Follow these steps to create a repository in GitHub

  1. Login to GitHub
  2. Click on the green button "New" or click on the "+"" in the top right corner then click on "create repository"
  3. On the new page type the name of the repository next to the owner name (yours)
  4. Check "Public", "add README file", "add .gitignore" (NODE), "choose a lisence" (any would work)
  5. Click "Create repository"
